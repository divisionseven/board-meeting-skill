# Test Suite

This directory contains the test suite for the Board Meeting Skill.

## Running Tests

Run all tests:
```bash
node test/suite.js
```

## Test Coverage

The test suite verifies:
- Install script syntax and execution
- All required files and directories exist
- All platform-specific files are present
- All templates are present
- All documentation files are present
- All GitHub templates exist
- All scripts exist
- All workflows exist

## Exit Codes

- `0` - All tests pass
- `1` - One or more tests fail
